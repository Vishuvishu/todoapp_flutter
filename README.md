## API Reference

#### Get all items


Visit This webspage to See Full documentation for CRUD 
https://api.nstack.in/#/Todo/TodoController_create


```http
  GET /https://api.nstack.in/v1/todos?page=1&limit=15
```



```http
  POST  /https://api.nstack.in/v1/todos
  request Body: 
  {
  "title": "string",
  "description": "string",
  "is_completed": false
}

```
